# 🌸 Mood Diary

A personal mood tracking app built with React. Track your daily emotions, sleep, and stress levels through an interactive calendar, and visualize your patterns on a dashboard.

## ✨ Features

- 📅 **Calendar** — Log your mood, stress level, sleep hours, and a personal note for each day
- 📊 **Dashboard** — Visual overview of your mood history, sleep patterns, and recent notes
- 💛 **Positivity** — Daily motivational quotes to brighten your day
- 🌸 **Custom profile** — Set your name and avatar in the header

## 🚀 Getting Started

### Prerequisites
- Node.js
- npm

### Installation

```bash
git clone https://github.com/bojananaumovska/mood-diary.git
cd mood-diary
npm install
npm start
```

Open [http://localhost:3000](http://localhost:3000) to view it in your browser.

## 🛠️ Built With

- [React](https://reactjs.org/)
- CSS Grid & Flexbox
- localStorage for data persistence
